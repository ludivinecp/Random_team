# Random_team
Generate a random team
```'rb'
# PERSONNALIZE TEAM
```

```'sh'
----- Joueur ? -----
$ Marie

----- Joueur ? -----
$ Lucie

----- Joueur ? -----
$ Pierre

----- Joueur ? -----
$ Laurent

----- Joueur ? -----
$ Anthony

----- Joueur ? -----
$ Coralie

```
When all you players have been added, tape :
```'sh'
$ .
```
Choose the number of team :

```'sh'
----- Number of team ? -----
$ 3
```

It will return 3 random teams  :

```'rb'
[["Laurent", "Anthony"], ["Lucie", "Coralie"], ["Marie", "Pierre"]]
```
